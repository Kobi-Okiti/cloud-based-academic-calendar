# Supabase Schema

This project uses a single SQL schema file for setup.

## Setup
1. Open your Supabase project.
2. Go to **SQL Editor**.
3. Run the file below:

```
/Users/kobi/Code/cloud-based-academic-calendar/supabase/schema.sql
```

## What it includes
- Tables, enums, and constraints
- RLS policies
- Triggers and helper functions (including `update_profile`)
- Notification rules and indexes

## Notes
- Run this once on a new project.
- If you already have data, review the schema before applying changes.
